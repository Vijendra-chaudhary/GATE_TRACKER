# GATE Tracker APK

This repository contains the Android APK file for the GATE CS Tracker application.

## Download

Download the latest APK: [app-debug.apk](app-debug.apk)

## Installation

1. Download the APK file
2. Enable "Install from Unknown Sources" in your Android settings
3. Install the APK on your device

## About

GATE Tracker is a comprehensive study companion for GATE CS exam preparation.
